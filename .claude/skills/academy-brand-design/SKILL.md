# AI Enablement Academy Brand Design Skill

Create professional, on-brand materials for AI Enablement Academy including presentations, training materials, marketing content, and visual resources.

## When to Use This Skill

Trigger this skill when the user requests:
- **Academy-branded** or **AEA branded** content
- Training presentations or workshop slides
- Course materials or learning resources
- LinkedIn posts or social media content
- Substack articles with visuals
- Marketing collateral or promotional materials
- Any visual content for AI Enablement Academy

## Core Brand Guidelines

### Brand Colors

**Primary Palette:**
- **Deep Blue** (Primary): `#1565C0` - Use for headers, CTAs, main brand elements
- **Light Blue** (Secondary): `#42A5F5` - Use for subheaders, links, accents
- **Dark Navy** (Text/Background): `#1A2332` - Primary text on light backgrounds, background for dark mode
- **Near White** (Background): `#F8F9FA` - Light mode backgrounds

**Spectrum Colors** (For variety, not hierarchy):
- Teal: `#4DB6AC`
- Green: `#81C784`
- Yellow: `#FDD835`
- Orange: `#FFB74D`
- Coral: `#FF7043`
- Magenta: `#EC407A`
- Purple: `#BA68C8`

**CRITICAL: Always use exact hex codes. No approximations.**

### Typography

**Font: Arial (ONLY) - Universal, accessible, professional**

**Type Scale:**
- **Display:** 60-72px, Bold, #1565C0 (light mode) or #42A5F5 (dark mode)
- **H1:** 48px, Bold, #1565C0/#42A5F5
- **H2:** 36px, Bold, #1565C0/#42A5F5
- **H3:** 24px, Semibold, #42A5F5
- **Body:** 16-18px (18px minimum for dark mode), Regular, #1A2332/#F8F9FA
- **Caption:** 14px, Regular, #5F6368

**Dark Mode Adjustments:**
- Increase body text to 18px minimum
- Line height: 1.8 (vs 1.6 in light mode)
- Use #F8F9FA text on #1A2332 background (13.8:1 contrast ratio)

### Dark Mode Default for Learning

**CRITICAL RULE:** All presentations, courses, and learning materials MUST use dark mode by default.
- **Background:** #1A2332 (Dark Navy)
- **Text:** #F8F9FA (Near White)
- **Headers:** #42A5F5 (Light Blue)
- **Exception:** Print materials only use light mode

**Why Dark Mode for Learning:**
- Reduces eye strain during extended viewing
- Better focus in presentation environments
- Enhances visual hierarchy
- Professional appearance in corporate settings

### Logo Usage

**Logo Rules:**
- **Minimum size:** 300px width (horizontal), 200px width (vertical), 48×48px (icon only)
- **Clear space:** 50% of logo height on ALL sides
- **Light mode:** Deep Blue (#1565C0) outline, Dark Navy (#1A2332) text
- **Dark mode:** Light Blue (#42A5F5) outline, White (#FFFFFF) text

**Placement:**
- Title slides: Top left or center
- Content slides: Small version in corner (consistent position)
- Marketing: Prominent position with clear space

### The Prism Icon - Strategic Usage

**CRITICAL: Prism icon frequency MAXIMUM 1 per 5 slides (20%)**

The prism represents transformation: Knowledge → Learning Process → Capabilities

**When to Use Prism Icon:**
- Key insight slides
- Theory→Practice bridges
- Transformation moments
- Memory anchors
- Assessment boundaries

**Prism Specifications:**
- **Size:** 60-80px (standalone), 48-60px (with label)
- **Position:** Top-center or left margin aligned with text
- **Always include label:** Uppercase, 10-12px, letter-spacing 0.15em
  - Label options: "KEY INSIGHT" | "REMEMBER THIS" | "APPLICATION" | "PRACTICE"
- **Color:** Use full color spectrum rays

**NEVER Use Prism For:**
- Decoration
- Every slide
- Bullet point markers
- Minor content
- Branding purposes

**Calculation:** Total slides ÷ 5 = Maximum prism uses

### Accessibility Requirements

**Contrast Ratios (WCAG):**
- **Minimum:** 4.5:1 for small text, 3:1 for large text (AA)
- **Preferred:** 7:1 for small text, 4.5:1 for large text (AAA)

**Pre-Approved Combinations:**
- Light mode: #1A2332 on #FFFFFF = 13.8:1 ✓
- Dark mode: #F8F9FA on #1A2332 = 12.6:1 ✓

**Accessibility Checklist:**
- Never use color as sole indicator
- Provide text alternatives for visuals
- Ensure keyboard navigation
- Use sufficient contrast ratios
- Keep line length ≤75 characters

## Content Type Guidelines

### Presentations & Training Materials

**Structure:**
1. **Title Slide:** Logo, course title (Display, #42A5F5), dark navy background
2. **Agenda Slides:** Numbered lists, spectrum colors for categorization
3. **Content Slides:** Dark mode, one main idea per slide, minimal text
4. **Section Dividers:** Bold large text (#42A5F5), minimal design
5. **Key Takeaways:** Consider prism icon if truly transformative

### LinkedIn Posts

**Visual Format:**
- **Size:** 1200×627px (landscape) or 1080×1080px (square)
- **Logo:** Include in corner with clear space
- **Typography:** Larger, bolder for social readability
- **Color:** Use primary blue (#1565C0) for brand recognition

### Substack Articles

**Header Images:**
- **Size:** 1600×900px recommended
- **Style:** Clean, professional, abstract or conceptual
- **Branding:** Subtle logo placement

## Quick Decision Guide

**Creating a presentation?** → Dark mode, Arial, spectrum colors, strategic prism use
**Creating a LinkedIn post?** → Light or dark user choice, bold typography, primary blue
**Creating a Substack header?** → Clean, conceptual, subtle branding
**Creating marketing materials?** → Light mode, strong CTAs, professional visuals
**Need to use prism icon?** → Max 1 per 5 slides, must be key moment

## Common Mistakes to Avoid

❌ **DON'T:**
- Use light mode for training presentations
- Approximate color codes
- Use fonts other than Arial
- Overuse prism icon (>20% of slides)
- Use prism icon decoratively
- Ignore contrast requirements
- Rotate, skew, or modify logo
- Crowd the logo (violate clear space)

✅ **DO:**
- Default to dark mode for learning
- Copy exact hex codes
- Use Arial exclusively
- Calculate prism frequency (slides ÷ 5)
- Use prism only for key moments
- Test contrast ratios
- Use provided logo files
- Maintain 50% clear space around logo

## Brand Voice & Tone

**Attributes:**
- **Clear:** Make complex AI concepts understandable
- **Confident:** Authoritative, evidence-based, professional
- **Empowering:** Enable action, not just awareness
- **Transformative:** Focus on capability change

**Writing Style:**
- Active voice
- Direct, specific language
- Avoid jargon unless audience-appropriate
- Professional but approachable
- Results-oriented
