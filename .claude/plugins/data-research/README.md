# Data Research Plugin

Enables compliant research, competitive analysis, and market scanning workflows.

## Included Assets
- **Agents**: `research-intelligence`, `competitive-analyst`, `market-scanner`.
- **Skills**: `web-scrape-safe`, `research-brief`, `/research:scan`.
- **Hooks**: compliance checker, rate-limit guard.

Integrates with Cloudflare browser-rendering MCP and OpenAI deep research sample server.
