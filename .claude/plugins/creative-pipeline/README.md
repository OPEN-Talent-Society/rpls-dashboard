# Creative Pipeline Plugin

Supports AI-first creative production across video, imagery, audio, and decks.

## Included Assets
- **Agents**: `creative-director`, `video-producer`, `image-curator`, `audio-editor`, `slides-producer`.
- **Skills**: `video-generate`, `image-variant`, `voiceover-transcribe`, `asset-approval`, `slides-from-outline`.
- **Hooks**: rights/compliance metadata, approval workflow notifications.

Use alongside creative asset library automations.
