# Workspace Automation Plugin

Automates communications, document sync, drive curation, and meetings across Google Workspace and Docmost.

## Included Assets
- **Agents**: `email-ops`, `doc-ingest`, `drive-curator`, `meeting-scribe`.
- **Skills**: `email-digest`, `doc-sync`, `drive-tag`, `meeting-notes`.
- **Hooks**: permissions audit, digest scheduling.
