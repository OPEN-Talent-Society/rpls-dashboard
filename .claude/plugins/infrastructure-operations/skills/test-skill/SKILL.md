# Test Skill

This skill demonstrates the proper Claude Code skill structure with subdirectories for scripts and templates.

## When to Use This Skill

Use this skill when you need to:
- Test basic infrastructure connectivity
- Run diagnostic scripts on services
- Verify file operations and permissions
- Test template generation capabilities

## Features

1. **Health Check Testing**: Run basic connectivity tests
2. **Script Execution**: Demonstrate script integration
3. **Template Usage**: Show how templates work
4. **File Operations**: Test file read/write capabilities

## Available Scripts

- `scripts/health-check.sh` - Basic service health testing
- `scripts/file-test.py` - File operation verification

## Templates

- `templates/test-report.md` - Standardized test report format

- "Run a basic health check on our services"
- "Test file operations in the project directory"
