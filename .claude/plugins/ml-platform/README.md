# ML Platform Plugin

Supports local ML stack operations (Whisper, Ollama), vector database (Qdrant), and RLHF lab workflows.

## Included Assets
- **Agents**: `local-ml-stack-ops`, `qdrant-ops`, `whisper-service-ops`, `ollama-orchestrator`, `rlhf-lab-coordinator`.
- **Skills**: `model-deploy`, `embedding-refresh`, `rl-eval-run`, `speech-transcribe`.
- **Hooks**: model release notifier, GPU utilisation alerts.
