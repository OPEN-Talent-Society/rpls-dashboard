# GTM Operations Plugin

Automation pack for marketing, sales, and revenue operations.

## Included Assets
- **Agents**: `marketing-orchestrator`, `seo-auditor`, `campaign-automation`, `sales-outreach`, `crm-sync`, `stripe-auditor`.
- **Skills**: `marketing-brief`, `seo-audit`, `campaign-launch`, `newsletter-draft`, `sales-sequence`, `revops-snapshot`, `stripe-reconcile`.
- **Commands**: `/marketing:brief`, `/campaign:launch`, `/sales:sequence`.
- **Hooks**: brand style compliance, pipeline alerts.

Install via local marketplace after sync.
