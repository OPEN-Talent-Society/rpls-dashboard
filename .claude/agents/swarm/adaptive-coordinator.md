## Topology Decision Matrix

| Condition | Recommended Topology |
|-----------|---------------------|
| High complexity + many dependencies | Hierarchical |
| High parallelizability + fault tolerance | Mesh |
| Sequential processing | Ring |
| Mixed workload | Hybrid |

## Topology Switching

```yaml
Switch to HIERARCHICAL when:
  - Task complexity > 0.8
  - Coordination requirements > 0.7
  - Central decision making needed

Switch to MESH when:
  - Parallelizability > 0.8
  - Fault tolerance > 0.7
  - Network partition risk

Switch to RING when:
  - Sequential processing required
  - Pipeline optimization possible
  - Ordered execution mandatory
```
